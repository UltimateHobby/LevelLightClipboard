Level Light Clipboard

Copy native Unreal Engine lighting setups between levels, save them as editable DataAssets, and paste only the modules you need.

Read documentation Latest release

Supported modules

  • Directional Light for sun
  • Optional Directional Light for moon
  • Sky Light
  • Sky Atmosphere
  • Exponential Height Fog
  • Volumetric Cloud
  • Post Process Volume
  • Selected World Settings fields

Why it exists

Reuse native lighting work without adopting an entire external sky or weather framework.

Editable assets

Lighting captures live in DataAssets, so artists can inspect and refine reusable presets directly.

Module by module

Copy or paste only atmosphere, fog, post process, sky lighting, or the full setup.

Explicit control

Every copy and paste dialog shows the actors involved before the tool changes a level.

Basic workflow

Designed to feel like lighting copy and paste, with the safety rails an editor tool needs.

  1. Open the source level.
  2. Open Tools -> Lighting Clipboard.
  3. Capture the modules you want into a clipboard DataAsset.
  4. Open the target level.
  5. Choose the same DataAsset and paste the selected modules.
Level Light Clipboard focuses on Unreal's native lighting actors. It does not replace them with a custom day-night or weather system, and it does not silently guess through ambiguous scenes.
