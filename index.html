Unreal Engine editor plugin

Level Light Clipboard

Version 1.0.0 UE 5.6 / 5.7 Windows

1. Summary

Level Light Clipboard copies native Unreal Engine lighting setups from one level into editable DataAssets, then pastes selected lighting modules into another level.

It is built for editor workflows where artists and level designers want to reuse a polished lighting setup without replacing Unreal's native lighting actors with a separate sky, weather, or time-of-day system.

2. Use Cases

  • Move a look-dev lighting setup into production maps.
  • Save reusable lighting presets as normal Unreal DataAssets.
  • Transfer only selected parts of a setup, such as fog, atmosphere, sky light, or post process.
  • Review source and target actors before the tool changes a level.

3. Basic Workflow

  1. Open the source level.
  2. Open Tools -> Lighting Clipboard.
  3. Copy the lighting modules you want into a Lighting Clipboard DataAsset.
  4. Open the target level.
  5. Select the same DataAsset.
  6. Paste only the modules you want to apply.

For step-by-step setup and troubleshooting, use the user guide.

4. Supported Modules

  • Sun / primary Directional Light
  • Optional moon / secondary Directional Light
  • Sky Light
  • Sky Atmosphere
  • Exponential Height Fog
  • Volumetric Cloud
  • Post Process Volume
  • Selected World Settings fields

5. Package Details

Product type Unreal Engine code plugin

Supported engine versions Unreal Engine 5.6 and 5.7
Target platform Windows / Win64
Blueprint-only projects Supported when using the packaged plugin build
Third-party dependencies None

6. Boundaries

  • The plugin focuses on Unreal's native lighting actors.
  • Ultra Dynamic Sky, Sky Creator, and project-specific environment rigs are not copied directly.
  • The plugin does not add a runtime day/night or weather system.
  • The plugin does not copy project settings, console variables, or destructive cleanup rules.
